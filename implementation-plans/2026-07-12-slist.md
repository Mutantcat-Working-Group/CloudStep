# `/slist` Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a public `GET /slist?way=...` (and `/slist/:way`) endpoint in CloudStep that returns the full URL list (id / parent / address / alive / retry) for a mapping set looked up by `way`, where `way` matches EITHER a self-help mode way OR a proxy mode way. No auth, cache-only read, matches `/self` / `/agent` conventions.

**Architecture:** One new read-path through the existing package-level cached maps in `collection/` (`SelfHelpMode`, `ProxyMode`, `WorkCllection`), accessed via a new `GetSelfHelpList(way)` helper. A new `SelfHelpListRouter` (implementing the existing `RouterTemplate` interface) registers `/slist` and `/slist/*name` with `Any(...)`. No schema change, no new DAO, no new package.

**Tech Stack:** Go 1.x, Gin v1.12.0, xorm v1.4.1, SQLite (mattn/go-sqlite3 v1.14.47), stdlib net/http/httptest for tests. No new dependencies.

## Working Context You'll Need

### Files in scope
- `collection/collection_store.go` - package-level global caches. Exported names (the typo is PART of the API; do NOT fix it here):
  - `WorkCllection map[string][]entity.Url` guarded by `MWorkCllection sync.Mutex`
  - `SelfHelpMode map[string]entity.SelfHelp` guarded by `MSelfHelpMode sync.Mutex`
  - `ProxyMode map[string]entity.Proxy` guarded by `MProxyMode sync.Mutex`
- `collection/self_help_mode.go` - `GetPath(way string) string` is the canonical locking pattern to copy (lock -> defer unlock -> read -> return).
- `collection/proxy_mode.go` - `GetProxyPath` / `GetProxyMode`: dual-cache-safe lookups.
- `entity/dao_entity.go` - `Url` has json tags `id`, `parent`, `address` (NOT `path`), `alive`, `retry`. Use exactly in tests.
- `router/router_template.go` - the `RouterTemplate` interface (PrepareRouter/InitRouter/DestroyRouter) every router must implement.
- `router/proxy_router.go` - closest template for a public read router (ProxyRouter).
- `router/self_help_router.go` - `/self` is the canonical "public, way -> cache read -> JSON" handler. Match error shape `{"code": 404}`.
- `router/login_router.go` - shows the LoginHandler() contrast; /slist does NOT use auth.
- `util/params_util.go` - `GetWayParam(c)` reads `way` query -> PostForm -> JSON body -> path param, in that order.
- `main.go` - registers all routers via `RegisterRouter(...)`; add the new router here.
- scheduler `hot_time_clean_scheduler.go` is unrelated (only clears login-hot-num).

### Patterns to follow
- Cache writes/reads always inside their guarded section.
- HTTP errors: `{"code": <int>, "msg": "<string>"}`. Success: `{"code": 0, "msg": "success", ...}`. Always `c.JSON(200, gin.H{...})`.
- Package-integer tests: `_test.go` in the same package, no testify needed.

### Patterns NOT to introduce (out of scope)
- No DB access, no new DAO, no new package, no migrations.
- Do NOT rename `WorkCllection` / `MWorkCllection` - the typo is preserved everywhere.
- Do NOT add filtering params, auth guards, or custom response envelopes beyond `gin.H`.

import (

	"reflect"
	"testing"

// resetCaches clears the three package-level maps in the initModes order so
// tests are hermetic. Tests exercise the cached in-memory maps only.
func resetCaches() {
	MWorkCllection.Lock()
	WorkCllection = make(map[string][]entity.Url)
	MWorkCllection.Unlock()

	MSelfHelpMode.Lock()
	SelfHelpMode = make(map[string]entity.SelfHelp)
	MSelfHelpMode.Unlock()

	MProxyMode.Lock()
	ProxyMode = make(map[string]entity.Proxy)
	MProxyMode.Unlock()
}

func TestGetSelfHelpList_SelfWay(t *testing.T) {
	resetCaches()
	const (
		way  = "alpha-self"
		coll = "alpha-collection"
	)
	urls := []entity.Url{
		{Id: 1, Parent: coll, Address: "http://a", Alive: true, Retry: 0},
		{Id: 2, Parent: coll, Address: "http://b", Alive: false, Retry: 1},
	}
	MWorkCllection.Lock(); WorkCllection[coll] = urls; MWorkCllection.Unlock()
	MSelfHelpMode.Lock(); SelfHelpMode[way] = entity.SelfHelp{Way: way, Point: coll}; MSelfHelpMode.Unlock()

	src, point, got := GetSelfHelpList(way)

	if src != SrcSelf {
		t.Fatalf("src = %v; want SrcSelf", src)
	}
	if point != coll {
		t.Fatalf("point = %q; want %q", point, coll)
	}
	if !reflect.DeepEqual(got, urls) {
		t.Fatalf("urls = %+v; want %+v", got, urls)
	}
}

## Task 2: Implement `GetSelfHelpList` in `collection` (TDD green)

**Files:** Create `collection/slist.go`. Test from Task 1 already on disk.

- [ ] **Step 1: Create `collection/slist.go`**

```go
package collection

import "com.mutantcat.cloud_step/entity"

// listSource classifies which in-memory mode-cache a way was resolved from.
type listSource int

const (
	SrcNone  listSource = iota // way not found in any mode cache
	SrcSelf                    // found in SelfHelpMode
	SrcProxy                   // found in ProxyMode
)

// String returns the wire value for the `mode` response field.
func (s listSource) String() string {
	switch s {
	case SrcSelf:
		return "selfhelp"
	case SrcProxy:
		return "proxy"
	default:
		return ""
	}
}

// GetSelfHelpList resolves `way` to its mapping set's URL list.
//
// Resolution order: SelfHelpMode, then ProxyMode. On collision the
// self-help entry wins (the two caches are not expected to share ways; we
// check self-help first in case a proxy way were accidentally duplicated).
//
// The way is treated as opaque. Each mode entry carries the target
// collection name in its `Point` field (== collection.Name == url.Parent).
// The returned `urls` slice is a direct reference into the package-level
// WorkCllection cache; callers MUST NOT mutate it.
//
// Returns (SrcNone, "", nil) when the way is unknown. Returns (src, point,
// nil) when the collection has zero urls.
func GetSelfHelpList(way string) (source listSource, point string, urls []entity.Url) {
	if way == "" {
		return SrcNone, "", nil
	}

	MSelfHelpMode.Lock()
	sh, ok := SelfHelpMode[way]
	MSelfHelpMode.Unlock()
	if ok {
		MWorkCllection.Lock()
		urls = WorkCllection[sh.Point]
		MWorkCllection.Unlock()
		return SrcSelf, sh.Point, urls
	}

	MProxyMode.Lock()
	px, ok := ProxyMode[way]
	MProxyMode.Unlock()
	if ok {
		MWorkCllection.Lock()
		urls = WorkCllection[px.Point]
		MWorkCllection.Unlock()
		return SrcProxy, px.Point, urls
	}

	return SrcNone, "", nil
}
```

Concurrency matches existing code: never hold more than one mutex at a time; lock -> single-map-read -> unlock -> lock-next. Returns a cache reference (matching `GetPath` / `GetProxyPath`), not a copy - this matches what `/self` already does and is safe because the only place the slice is mutated is the admin-write codepath, which is exclusive.
